# Employee Attrition Analysis Using Excel 📊

## 📌 Project Overview
This project focuses on analyzing employee attrition using Microsoft Excel. The goal is to understand the key factors that contribute to employees leaving the organization and to present meaningful insights using dashboards, pivot tables, and charts.

By converting raw HR data into visual reports, this project helps in identifying patterns related to age, department, overtime, job role, and tenure.

---

## 🎯 Objectives
- To analyze employee attrition trends
- To identify key factors affecting employee turnover
- To create interactive dashboards using Excel
- To present insights in a simple and visual format

---

## 📂 Dataset Information
- Total Records: 1470 Employees  
- Features include:
  - Age
  - Department
  - Job Role
  - Monthly Income
  - Overtime
  - Years at Company
  - Attrition Status

---

## 🛠️ Tools Used
- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - KPI Metrics
  - Dashboard Design

---

## 📊 Key Insights
- Employees working overtime show higher attrition rates
- Certain departments have higher turnover than others
- Younger employees are more likely to leave
- Low income groups show higher attrition probability

---

## 📈 Dashboard Features
- Interactive filters (Slicers)
- KPI cards (Attrition Rate, Total Employees, Active Employees)
- Visual charts for department-wise and age-wise analysis
- Clean and simple UI for better understanding
